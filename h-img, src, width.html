상세설명 - 생활코딩: 최후의 문법 속성과 img

이미지 img

이미지를 넣는 태그의 이름은 img입니다. image의 줄임말이겠죠? 하지만 뭔가 좀 허전합니다. 태그 이름만으로는 어떤 이미지를 가져올 지 정보가 부족하다는 것입니다. 그래서 새로운 문법을 도입하게 됩니다. 속성(attribute)입니다. 속성을 적용해 봅시다.

  1. 속성: 이미지 불러오기 src

    2. img태그에 source의 줄임말인 src를 붙여서 코드를 이와 같이 만듭니다. img src="이미지의 주소" 이렇게 src라는 속성을 통해서 이미지를 불러올 수 있게 된 것입니다.
    그럼 내 컴퓨터에 있는 이미지를 표시하고 싶다면 어떻게 해야할까요? 웹브라우저에서 원하는 이미지를 찾은 다음에 원하는 위치에 이미지를 저장하시면 됩니다.
    다운로드 받은 이미지를 프로젝트 폴더에 위치시켜 보세요. 웹페이지 파일과 같은 위치에 있는 이미지 파일이라면 src 속성의 값으로 이미지 파일의 이름만 적어주시면 됩니다.
    img src="이미지 파일 이름"

  2. 속성: 이미지 크기 조정 width

    3. 이미지의 크기가 원하는 것과 다르면 어떻게 하면 될까요? 속성 width의 값으로 숫자나 %를 사용하면 원하는 크기로 조정할 수 있습니다.
    img src="이미지 파일 이름 or 주소" width= "100%"


  • 위에서 살펴본 바와 같이 속성은 태그의 이름만으로는 정보가 부족할 때 사용됩니다.
