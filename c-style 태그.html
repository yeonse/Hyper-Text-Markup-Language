생활코딩: 혁명적 변화

HTML과 CSS는 완전히 다른 언어이기 때문에 웹페이지에 CSS를 포함시키기 위해서는 웹브라우저로 하여금 어디서부터 어디까지가 CSS인지를 알 수 있게 지정해줘야 됩니다.
그 방법으로는 html의 style 속성, css의 style 태그 이 두가지를 이용하는 방법이 있습니다.

style 태그

style태그 안에 적은 코드들은 웹브라우저에게 CSS언어라고 인지시키고 그 문법에 맞게 해석하도록 명령합니다.

예시 코드 : style a{color:red;} /style 분석

  • style 태그

    • - 효과를 누구에게 지정할지를 설명할 수가 없기 때문에 이 style태그 안에 a { } 라고 하는 이런 코드가 추가적으로 더 필요한 겁니다.

  • a

    • - 이 웹 페이지의 모든 a라는 태그에게 어떠한 효과를 준다. 즉, "주고 싶은 효과를 누구에게 줄 것인가를 선택하다" 라는 점에서 저것을 선택자Selector라고 부릅니다.

  • color:red

    • - 선택자가 지정하는 태그들에 대해서 어떤 효과를 줄 것인지에 해당하며 선언Declaration 또는 효과라고 부릅니다.

  • color

    • - 속성Property라고 부르며,

  • red

    • - 속성의 값Value라고 부릅니다. 효과 끝에는 ;세미콜론을 꼭 붙여주어야 합니다.
