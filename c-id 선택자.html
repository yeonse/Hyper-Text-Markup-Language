id 선택자

id선택자도 클래스 선택자와 마찬가지로 웹 문서 안의 특정 부분에 스타일을 지정할 때 사용하며 변경하고 싶은 태그 내에 id코드를 작성하고 이름을 설정합니다.

클래스 선택자의 경우 문서 안에서 여러 번 적용할 수 있는 스타일인 반면, id 선택자는 문서 안에서 한 번만 적용할 수 있다는 것입니다. id 선택자는 중복 사용이 불가하기에 주로 문서의 레이아웃과 관련된 스타일을 지정하거나 자바스크립트 프로그램에서 웹 요소를 구분하기 위해 자주 사용합니다.
