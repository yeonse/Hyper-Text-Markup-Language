CSS

CSS는 Cascading Style Sheet의 약어로 html과는 완전히 다른 컴퓨터 언어입니다. 웹을 만든 사람들은 웹페이지를 아름답게 만드는 방법에 대한 깊은 고민에 빠집니다. 고민 끝에 두가지 갈림길에 서게 됩니다. 하나의 길은 기존 html언어에 디자인에 대한 새로운 태그를 추가하는 것이였습니다. 하지만 그 한계를 깨닫고 곧 어렵지만 근본적인 해결책을 찾아나섰습니다. 그 해결책은 CSS라는 새로운 언어를 만들게 된 것입니다.

마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있습니다. 마크업 언어(ex: HTML)가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다고 할 수 있습니다. 즉, HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 웹사이트처럼 꾸밀 수 있습니다.
